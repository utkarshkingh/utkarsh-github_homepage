## 🚀 Projects

### 🛡️ [Real-Time AI Financial Fraud Detection](https://github.com/utkarshkingh/AI-Financial_FraudDetection)
An end-to-end system that catches fraudulent transactions as they happen. Transactions stream in through Kafka, an XGBoost model scores them in real time, and the whole pipeline retrains itself daily — so it keeps up as fraud patterns shift. Detects 4 fraud types: account takeover, card testing, merchant collusion, and geographic anomalies.
`Kafka` · `Spark` · `XGBoost` · `MLflow` · `Airflow` · `Docker`

### 🤖 [Agentic Trading Bot](https://github.com/utkarshkingh/agentic-trading-bot)
A team of AI agents that make trading decisions together. They pull live market data straight from a stock broker (via a Zerodha MCP server), then run technical and sentiment analysis to reach a coordinated call — and it works with any LLM provider.
`Agentic AI` · `MCP` · `LangGraph` · `FastAPI` · `LiteLLM`

### 📄 [LLM Fine-tuning: Document Intelligence](https://github.com/utkarshkingh/bpmncoder_llama3.2-vision_finetuned)
Taught a vision-language model (Llama 3.2 Vision) to turn process diagrams into structured code automatically, and compared different training methods (standard fine-tuning vs. RLHF/DPO) to see which makes the model more reliable.
`Hugging Face` · `PyTorch` · `LoRA` · `RLHF/DPO`

### 🧠 Brain-Signal Movement Detection — *FAU N² Lab*
Read brain signals to control a prosthetic leg, classifying movement intent fast and accurately (94% F1, 30ms response) so the device reacts almost instantly.
`PyTorch` · `Signal Processing` · `scikit-learn`
